# 金融主题表情包制作项目

## 项目信息

- **项目名称**：金融主题表情包（韭菜日记系列）
- **制作方案**：A方案（小韭菜）+ B方案（小动物牛/熊）混合
- **表情数量**：24个
- **格式**：静态PNG（16个）+ 动态GIF（8个）
- **开始日期**：2025年12月12日

## 一、角色设计方案

### 主角色1：小韭菜（方案A）
**形象设定**：
- 外形：Q版韭菜拟人，绿色头发（韭菜叶），圆润身体
- 特征：大眼睛、小嘴巴、可爱表情
- 配色：翠绿色（#00CC66）+ 白色 + 粉色腮红
- 性格：乐观、自嘲、可爱、有点傻萌
- 用途：用于大部分表情（约14-16个）

### 主角色2：小牛/小熊（方案B）
**小牛形象**（牛市相关）：
- 外形：Q版小牛，圆滚滚
- 特征：向上的牛角、坚定的眼神
- 配色：红色（#FF4444）+ 金色
- 用途：涨势相关表情（约4-5个）

**小熊形象**（熊市相关）：
- 外形：Q版熊，憨厚
- 特征：圆耳朵、无辜眼神
- 配色：深绿色（#006644）+ 棕色
- 用途：跌势相关表情（约4-5个）

### 辅助元素
- K线图（红绿蜡烛）
- 涨停/跌停板
- 钱袋、钞票
- 手机/电脑屏幕
- 箭头（上涨↑、下跌↓）
- 表情符号（❤️💔😭😄）

## 四、技术规范

### 文件规格
```
静态表情（PNG）：
- 尺寸：240 x 240 px
- 分辨率：72 DPI
- 色彩模式：RGB
- 背景：透明
- 格式：PNG-24
- 大小：< 150KB

动态表情（GIF）：
- 尺寸：240 x 240 px
- 分辨率：72 DPI
- 色彩模式：RGB（256色）
- 背景：透明
- 格式：GIF
- 帧数：8-12帧
- 循环：无限循环
- 大小：< 300KB
```

### 设计规范
```
色彩方案：
- 主色-红（涨）：#FF4444, #FF6B6B
- 主色-绿（跌）：#00CC66, #00AA55
- 主色-金（钱）：#FFD700, #FFA500
- 辅助色：#FFFFFF, #333333, #F5F5F5

字体：
- 中文：思源黑体 Bold / 阿里巴巴普惠体 Bold
- 英文：Arial Bold / Helvetica Bold
- 字号：24-32px
- 描边：白色2px（保证可读性）

风格要求：
- 线条粗细：3-4px
- 圆角：统一使用圆润处理
- 阴影：使用柔和投影
- 表情：夸张但不失真
```

## 八、注意事项

### 设计注意
- ✅ 表情要够夸张，一眼看懂情绪
- ✅ 颜色不要过于鲜艳刺眼
- ✅ 保证在手机小屏幕上清晰可见
- ✅ 透明背景要处理干净
- ✅ 动态GIF循环要自然

### 内容注意
- ❌ 不涉及具体股票代码
- ❌ 不包含违规诱导内容
- ❌ 不使用真实人物形象
- ❌ 不包含敏感政治经济话题
- ✅ 幽默但不过度负面
- ✅ 自嘲风格，接地气

### 审核要点
- 符合微信表情审核规范
- 无侵权风险
