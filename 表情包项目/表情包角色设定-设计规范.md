# 金融表情包 - 角色设定设计规范

## 主角色1：小韭菜 🌱

### 基础形象设定

#### 整体造型
```
身高比例：2-2.5头身（Q版比例）
体型：圆润、可爱、略带婴儿肥
风格：简约、现代、亲和力强
```

#### 头部设计
- **韭菜叶发型**：
  - 3-5片韭菜叶向上生长
  - 颜色：翠绿色（#00CC66）→ 浅绿色（#66E6A3）渐变
  - 叶片顶端略微弯曲，增加动感
  - 发根处是白色/浅黄色（#FFFEF0）

- **脸型**：
  - 圆形，下巴圆润
  - 腮红：粉色圆形（#FFB6C1），位于脸颊两侧
  - 肤色：象牙白（#FFFFF0）或浅米色（#FFF8DC）

- **眼睛**（重要！表情核心）：
  - 大眼睛，圆形或椭圆形
  - 黑色眼珠，带高光点（增加灵动感）
  - 可以有简单的睫毛（3-4根）
  - 眉毛：细线条，位于眼睛上方，可根据表情变化角度

- **嘴巴**：
  - 小嘴，简洁线条
  - 开心时：W形或U形
  - 伤心时：倒U形
  - 惊讶时：O形

#### 身体设计
- **躯干**：
  - 圆润的椭圆形或梨形
  - 颜色：浅绿色（#E8F5E9）或白色
  - 可选：穿小背心或T恤（白色或浅色）

- **手臂**：
  - 短小圆润
  - 4指手掌（卡通风格）
  - 可做各种动作（指点、挥手、抱头等）

- **腿脚**：
  - 短腿，圆润
  - 脚掌简化为圆形或椭圆形
  - 可选：穿小鞋子（白色运动鞋）

### 表情变化参考

#### 开心表情系列
1. **大笑**：眉毛上扬，眼睛眯成月牙，嘴巴大大的W形
2. **微笑**：眉毛平缓，眼睛正常，嘴角上扬
3. **得意**：眉毛挑起，眼睛一眯，嘴角歪笑
4. **兴奋**：眉毛上扬，眼睛睁大，嘴巴张大O形

#### 伤心表情系列
1. **大哭**：眉毛下垂，眼睛紧闭，泪珠飞溅，嘴巴倒U形
2. **委屈**：眉毛下垂，眼睛含泪，嘴巴扁扁的
3. **绝望**：眉毛下垂，眼睛无神（白眼），嘴巴张开
4. **郁闷**：眉毛皱起，眼睛半闭，嘴巴抿着

#### 惊讶表情系列
1. **震惊**：眉毛上扬，眼睛圆睁，嘴巴O形
2. **疑惑**：眉毛皱起，眼睛斜视，嘴巴歪着
3. **懵逼**：眉毛平直，眼睛呆滞（螺旋形），嘴巴微张

#### 其他表情
1. **愤怒**：眉毛倒八字，眼睛瞪圆，嘴巴咬牙
2. **淡定**：眉毛平缓，眼睛半闭，嘴巴一条线
3. **无奈**：眉毛下垂，眼睛半闭，嘴角下垂

### 动作姿势参考

#### 基础姿势
- **站立**：双手自然下垂或背后，双腿并拢
- **坐姿**：盘腿坐或侧坐
- **躺平**：仰面朝天，四肢伸展

#### 手部动作
- **挥手**：高举右手摆动
- **指点**：单手指向前方
- **抱头**：双手抱住头部
- **捂脸**：双手遮住脸
- **竖大拇指**：单手大拇指向上
- **双手托腮**：坐着，双手撑脸
- **捧东西**：双手在胸前捧着（钱、手机等）

#### 动态动作
- **跳跃**：双腿弯曲蓄力 → 离地 → 落地
- **奔跑**：前倾，双腿一前一后
- **摔倒**：身体倾斜 → 倒地 → 躺平
- **飞起**：双腿离地，向上升

## 主角色2：小牛 🐮（牛市代表）

### 基础形象设定

#### 整体造型
```
身高比例：2头身
体型：壮实、圆润、有力量感
风格：积极向上、充满能量
```

#### 头部设计
- **牛角**：
  - 两个向上弯曲的角
  - 颜色：米黄色（#F5DEB3）
  - 角尖可以发光（表示牛气冲天）

- **脸型**：
  - 圆形，略带方形（体现力量）
  - 肤色：奶黄色（#FFE4B5）或浅红色（#FFCCCB）

- **眼睛**：
  - 圆眼睛，眼神坚定
  - 可以有闪光效果
  - 眉毛：粗壮，向上扬

- **鼻子**：
  - 牛鼻子，两个大鼻孔
  - 颜色：深色（#8B4513）

- **耳朵**：
  - 圆形垂耳，位于头顶两侧
  - 颜色：与脸部同色，内耳浅色

#### 身体设计
- **躯干**：
  - 健壮的圆柱形
  - 颜色：红色（#FF4444）或金黄色（#FFD700）
  - 可以有花纹或渐变

- **四肢**：
  - 粗壮有力
  - 牛蹄：黑色或深棕色
  - 可做奔跑、站立动作

- **尾巴**：
  - 细长尾巴，尾尖有毛簇
  - 可以甩动（动态表情）

### 表情变化
- **兴奋**：眼睛瞪大，鼻孔喷气，嘴角上扬
- **冲刺**：眉头紧锁，眼神凌厉，咬牙切齿
- **得意**：眼睛微眯，嘴角上扬，头部高昂
- **疲惫**：眼睛半闭，气喘吁吁，头部低垂

### 动作姿势
- **冲刺**：四腿弯曲，身体前倾，准备冲出
- **飞奔**：四腿张开，扬起灰尘
- **昂首**：站立，头部高昂，角向天空

## 主角色3：小熊 🐻（熊市代表）

### 基础形象设定

#### 整体造型
```
身高比例：2头身
体型：圆滚滚、憨厚、略显笨重
风格：憨态可掬、无辜、有点丧
```

#### 头部设计
- **整体**：
  - 圆形大头
  - 颜色：深绿色（#006644）或深棕色（#654321）

- **耳朵**：
  - 圆形小耳朵，位于头顶两侧
  - 颜色：比脸部稍深

- **脸型**：
  - 圆润，肉乎乎
  - 有白色或浅色的口鼻部（#F5F5DC）

- **眼睛**：
  - 小圆眼，眼神无辜或呆滞
  - 可以有黑眼圈（表示熬夜看盘）

- **鼻子**：
  - 圆形黑鼻子（#000000）

- **嘴巴**：
  - 小嘴，表情多为下垂或平淡

#### 身体设计
- **躯干**：
  - 圆滚滚，略显臃肿
  - 颜色：绿色（#00AA55）
  - 肚子部分可以是浅色

- **四肢**：
  - 短粗，圆润
  - 熊掌：大大的，有肉垫
  - 动作多为缓慢、笨拙

- **尾巴**：
  - 小短尾，毛茸茸

### 表情变化
- **无辜**：眼睛圆睁，嘴巴微张，一脸懵
- **压迫**：眼睛微眯，嘴角下垂，气势汹汹
- **疲惫**：眼睛半闭，嘴巴张开，打哈欠
- **呆滞**：眼睛无神，嘴巴一条线

### 动作姿势
- **压迫**：身体庞大，双掌按压小韭菜
- **站立**：双脚站立，双手抱胸
- **坐着**：盘腿坐地，显得笨重
- **趴着**：四肢趴地，一脸疲惫

## 字体设计规范

### 表情配文字体

#### 中文字体推荐
```
首选：思源黑体 Bold / Source Han Sans Bold
备选1：阿里巴巴普惠体 Bold
备选2：站酷快乐体（更活泼）
备选3：方正兰亭黑 Bold
```

#### 字体参数
```
字号：28-36px（根据文字长度调整）
字重：Bold（加粗）
颜色：白色（#FFFFFF）
描边：3-4px，颜色为深色（#333333）或与背景反差色
位置：通常在表情下方或上方
对齐：居中对齐
```

#### 文字排版规则
```
- 短文字（2-4字）：可以更大，36px
- 中等文字（5-8字）：32px
- 长文字（9-12字）：28px，可分两行
- 超长文字：不建议，简化表达
```

### 特殊文字效果

#### 强调效果
```
放大：关键字可以放大1.2-1.5倍
颜色：可以用红色或金色突出
符号：加感叹号、省略号增强情绪
```

#### 动态效果（仅GIF）
```
闪烁：重要文字闪烁出现
晃动：左右轻微摆动
渐显：逐字出现
```

## 线条与描边规范

### 线条粗细
```
主线条（轮廓）：3-4px
细节线条（五官）：2-3px
装饰线条（纹理）：1-2px
```

### 描边处理
```
角色描边：深色（#333333），3px
元素描边：根据主色调，2-3px
文字描边：深色（#333333）或反差色，3-4px
```

### 圆角处理
```
统一使用圆润处理：
- 大元素圆角：10-15px
- 小元素圆角：5-8px
- 所有尖角都要圆滑化
```

## 动画设计规范（GIF专用）

### 帧数设定
```
简单动画：8-10帧
中等动画：10-12帧
复杂动画：12-15帧（尽量不超过）
```

### 帧率
```
推荐：10-12fps
快速动作：15fps
慢速动作：8fps
```

### 循环设定
```
循环方式：无限循环
首尾衔接：最后一帧要能自然过渡到第一帧
停顿：可在关键帧停顿2-3帧增强效果
```

### 动画时长
```
总时长：1-2秒
快速动作：0.8-1.2秒
慢速动作：1.5-2秒
```

### 缓动原则
```
启动：慢速启动（easing in）
结束：慢速结束（easing out）
中间：快速过渡
```

### 关键帧设计

#### 跳跃动画（8帧示例）
```
帧1：蹲下蓄力
帧2：蹲得更低
帧3：开始离地
帧4：升至最高点（停顿2帧）
帧5：开始下落
帧6：快速下落
帧7：着地
帧8：站立恢复
```

#### 哭泣动画（10帧示例）
```
帧1-2：眼睛睁开
帧3-4：眼睛挤压，泪水出现
帧5-6：泪水滴落（停顿）
帧7-8：泪水继续落
帧9-10：循环回到帧3
```

#### 闪烁动画（8帧示例）
```
帧1-2：正常
帧3-4：发光元素出现并放大
帧5-6：光效最强（停顿）
帧7-8：光效减弱消失
```

## 导出规范

### 静态PNG设置
```
格式：PNG-24
尺寸：240 x 240 px
分辨率：72 DPI
色彩模式：RGB
背景：透明
压缩：PNG优化（保证<150KB）
```

### 动态GIF设置
```
格式：GIF
尺寸：240 x 240 px
分辨率：72 DPI
色彩模式：RGB，256色
背景：透明
循环：无限循环
抖动：适中（防止色阶断裂）
压缩：GIF优化（保证<300KB）
```

### 文件命名规范
```
格式：序号-名称-类型.扩展名
示例：
01-涨停了-动态.gif
09-盈利了-静态.png
```
